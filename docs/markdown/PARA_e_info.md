### 组队
每6人一组，如不能归整，余数分别自由组到已有组别。每组自行推选一个组长，组名自选任何一种动物名。

推选课程代表1-2人，负责课程事宜。

### 考核

* 平时成绩

1. 小组实验，组内成员贡献度由小组根据个人贡献大小共同确定（需诚实守信）， 百分比计算，综合为100%。
2. 签到，由课代表及组长负责（需诚实守信）。

* 期末成绩
