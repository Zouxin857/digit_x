# 该章讲述如何利用grasshopper，将装置测量到的数据进行可视化表现；

## 思路：将kml 转化为曲线；

         将曲线以记录时间节点打断；

         将每段生成面（mesh）；

         将颜色与excel 图表中数据一一对应，填充到面上  

重点知识涉及到**bumblebee插件将excel表格导入，以及list数列、tree数列的灵活处理**。
